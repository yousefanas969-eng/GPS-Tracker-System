# GPS-Tracker-System
A real-time GPS tracking system that monitors live coordinates and visualizes movement direction with a dynamic heading arrow 

## Hardware Used
* **Microcontroller:** Arduino
* **Display:** OLED SH1106 (128x64)
* **GPS Module:** Neo-6M (or compatible)

## Libraries Used
* `TinyGPS++`
* `U8g2lib`
* `SoftwareSerial`
